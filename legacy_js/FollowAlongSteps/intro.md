The Follow Along Steps files contain all the commands you need to complete certain actions. They're designed to be used in VS Code.

They're designed that you can run each function in the correct sequence, updating the relevant variables as you go.

Hints:

The usage is simple. In each step, after updating any relevant variables, select all the code in the block and Run it by pressing Ctrl + Alt + N. 

If the guide instructs you to comment/uncomment certain lines, you can do this by highlighting those lines and entering Ctrl + /

Note that in order to do this, you will need to have Node.JS installed plus all of the packages relevant to this project. You will also need the Code Runner extension.

Before running any of these, you will want to ensure you choose an appropriate download folder in config/read_write_settings/readWriteSettings.yaml
